## 🚀 快速开始

### 在线使用
直接访问 [https://1850741061.github.io/life-manager/](https://1850741061.github.io/life-manager/) 即可使用，无需安装。

### 桌面版安装
前往 [Releases](https://github.com/1850741061/life-manager/releases) 下载最新版安装包（Windows），运行安装即可。

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/1850741061/life-manager.git
cd life-manager

# 安装依赖
npm install

# 启动开发
npm start

# 打包安装包
npm run build:win
```

## 📖 使用指南

### 基础任务管理

#### 创建任务
1. 在输入框中输入任务标题
2. 选择日期、优先级、分组/项目
3. 点击"添加"按钮或按 Enter 键

#### 管理任务
- **完成任务**: 点击任务左侧的圆圈
- **编辑任务**: 点击任务旁的编辑图标
- **删除任务**: 使用确认框删除
- **添加子任务**: 打开任务详情，添加子任务

### 侧边栏筛选
- **全部任务**: 查看所有非项目任务
- **今日**: 聚焦今天到期的任务
- **超时任务**: 查看所有逾期未完成的任务
- **分组**: 按自定义分组筛选

### 桌面小组件
1. 点击工具栏的小组件按钮启动
2. 任务面板显示今日待办，底部折叠区显示超时任务
3. 项目面板点击展开查看任务详情
4. 在小组件中操作任务会自动同步到主应用，反之亦然

### 云端同步
1. 点击工具栏的云图标
2. 注册新账号或登录现有账号
3. 数据自动同步到云端，多设备实时同步

## 🛠 技术栈

- **前端**: 纯 HTML/CSS/JavaScript (ES6+)
- **桌面端**: Electron 28
- **UI 风格**: 自定义波普艺术风格
- **数据存储**: LocalStorage + Supabase
- **图表库**: Chart.js
- **图标库**: FontAwesome 6
- **字体**: Google Fonts (Inter)
- **打包工具**: electron-builder
