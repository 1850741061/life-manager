## 🚀 快速开始

### 在线使用
直接访问 [https://1850741061.github.io/life-manager/](https://1850741061.github.io/life-manager/) 即可使用，无需安装。

### 本地部署

#### 方式一：直接使用
```bash
# 克隆仓库
git clone https://github.com/1850741061/life-manager.git
cd life-manager

# 直接在浏览器中打开 index.html
# 或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

#### 方式二：使用GitHub Pages（推荐）
1. Fork本仓库到你的GitHub账号
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为发布源
4. 访问 `https://你的用户名.github.io/life-manager/`

#### 方式三：打包为桌面应用
```bash
# 安装Electron
npm install -g electron
npm install -g electron-packager

# 打包Windows应用
electron-packager . ProLife --platform=win32 --arch=x64

# 打包macOS应用
electron-packager . ProLife --platform=darwin --arch=x64
```

## 📖 使用指南

### 基础任务管理

#### 创建任务
1. 在输入框中输入任务标题
2. 选择日期、优先级、分组/项目
3. 点击"添加"按钮或按Enter键

#### 管理任务
- **完成任务**: 点击任务左侧的圆圈
- **编辑任务**: 点击任务旁的编辑图标
- **删除任务**: 使用自定义确认框删除
- **添加子任务**: 打开任务详情，添加子任务

### 项目管理 ⭐

#### 创建项目
1. 点击工具栏的"项目管理"按钮（或侧边栏"管理项目"）
2. 点击"添加新项目"
3. 输入项目名称、描述、颜色、截止日期
4. 保存项目

#### 为项目添加任务
1. 点击侧边栏的项目名称
2. 创建任务，自动归属到当前项目
3. 项目任务使用紧凑卡片样式，节省空间

#### 查看项目统计
1. 点击工具栏"数据统计"按钮
2. 滚动到项目列表区域
3. 每个项目显示：
   - 项目名称和描述
   - 统计卡片（总任务/已完成/进行中/完成率）
   - 进度条
   - 饼图（已完成 vs 进行中）

#### 删除项目
- 删除有任务的项目时，会弹出三选项：
  - **取消删除**: 不做任何操作
  - **删除项目下的所有任务**: 永久删除项目及其任务
  - **保留任务并移至默认分组**: 将任务移至第一个分组

### 日历视图

#### 查看任务分布
- **彩色圆点**: 表示分组任务（按分组颜色）
- **彩色方块**: 表示项目任务（按项目颜色）
- **点击日期**: 查看当日的任务详情面板
- **项目卡片**: 点击展开查看该项目的所有任务

### 番茄钟

#### 使用番茄钟
1. 点击工具栏的番茄钟图标
2. 点击"开始"启动25分钟倒计时
3. 工作结束后自动进入休息模式
4. 可随时暂停或重置

### 记账功能

#### 记录收支
1. 切换到"记账"视图
2. 点击"添加记录"
3. 选择类型（收入/支出）、分类、金额
4. 保存记录

#### 查看统计
- 切换到"月份视图"
- 查看每月的总收入、总支出和结余
- 查看各分类的支出占比饼图

### 云端同步

#### 开启云同步
1. 点击工具栏的云图标
2. 注册新账号或登录现有账号
3. 数据自动同步到云端

#### 多设备使用
1. 在其他设备上打开应用
2. 使用同一账号登录
3. 数据自动同步到本地

## 🛠 技术栈

- **前端框架**: 纯HTML/CSS/JavaScript (ES6+)
- **UI框架**: 自定义波普艺术风格
- **数据存储**: LocalStorage + Supabase
- **图表库**: Chart.js
- **图标库**: FontAwesome 6
- **字体**: Google Fonts (Inter)
- **构建工具**: 无需构建，开箱即用
